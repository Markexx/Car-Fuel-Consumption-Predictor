# Auto MPG Predictor

## Opis problema
Predviđanje potrošnje goriva automobila (MPG) na temelju tehničkih karakteristika.

## Tehnologije
- Python 3.13
- Flask API
- Scikit-learn
- HTML/JavaScript

## Pokretanje
1. pip install -r requirements.txt
2. python train.py
3. python app.py
4. Otvorite http://localhost:5001
